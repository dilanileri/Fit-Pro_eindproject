# Fit-Pro

Fitness webapplicatie voor een sportschool.

## Rollen
- Gast
- Member
- Admin

## Functionaliteiten
- Trainingsschema’s per doel
- Workouts met oefeningen
- Oefening detailpagina met uitleg, foto en of video
- Profielpagina met BMI berekening
- Abonnementensysteem
- Adminbeheer voor trainingsschema’s en oefeningen
