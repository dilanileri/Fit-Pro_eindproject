# Fit-Pro

Fitness webapplicatie ontwikkeld als eindwerk Full Stack Web Development.

Fit-Pro helpt sporters met het volgen van trainingsschema’s, workouts en oefeningen binnen een moderne en gebruiksvriendelijke fitnessomgeving.

---

## Features

### Demo Betalingssysteem
-registreren mogelijk!

### Member Functionaliteiten
- Trainingsschema’s bekijken per doel
- Workouts met gekoppelde oefeningen
- Oefening detailpagina’s met uitleg, afbeeldingen en video’s
- Favorieten systeem voor trainingsschema’s
- BMI berekening op profielpagina
- Membership checkout flow
- Responsive premium UI

### Admin Functionaliteiten
- Beheer van trainingsschema’s
- CRUD voor workouts
- CRUD voor oefeningen
- Uploaden van afbeeldingen
- Relaties tussen schema’s, workouts en oefeningen beheren
